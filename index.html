📥 Importar expediciones desde Excel

⚙ Conectar con Google Sheets

Sigue estos pasos una sola vez (solo el administrador necesita hacerlo):

  1. Ve a sheets.google.com y crea una hoja llamada ParteCargas
  2. Menú: Extensiones → Apps Script
  3. Borra el código de ejemplo y pega el contenido del archivo apps-script.gs
  4. Clic en Implementar → Nueva implementación → Tipo: Web App
  5. En "Quién tiene acceso" elige Cualquier usuario → clic en Implementar
  6. Copia la URL y pégala aquí abajo. ¡Listo! Todos los PCs solo necesitan esta URL.
